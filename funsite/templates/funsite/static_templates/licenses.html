Conditions d’utilisation

Les contenus diffusés sur la plateforme FUN sont régis par quatre types de licence soumises au droit français :

  • Licence restrictive : l’utilisateur ne peut exploiter l’œuvre qu’à des fins personnelles et doit mentionner le nom de l’auteur ;
  • Licence CC BY NC ND : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial et il ne peut apporter de modifications de l’œuvre originale ;
  • Licence CC BY NC : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial, il peut créer une œuvre dérivée (modifications) de l’œuvre originale ;
  • Licence CC BY NC SA : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial, il peut créer une œuvre dérivée de l’œuvre originale si l’œuvre dérivée est diffusée sous la même licence que l’œuvre originale.

En participant à un cours en ligne ouvert et massif, vous disposez également de droits quant à l’utilisation du contenu que vous serez amené à produire. Par défaut, votre production relève de votre propriété intellectuelle et ne peut donc pas être réutilisée sauf mention contraire sur le descriptif du cours. Dans ce dernier cas, trois types de licences Creative Commons peuvent s’appliquer sur votre production :

  • Licence CC BY NC ND : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial et il ne peut apporter de modifications de l’œuvre originale ;
  • Licence CC BY NC : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial, il peut créer une œuvre dérivée (modifications) de l’œuvre originale ;
  • Licence CC BY NC SA : l’utilisateur doit mentionner le nom de l’auteur, il peut exploiter l’œuvre sauf dans un contexte commercial, il peut créer une œuvre dérivée de l’œuvre originale si l’œuvre dérivée est diffusée sous la même licence que l’œuvre originale.
